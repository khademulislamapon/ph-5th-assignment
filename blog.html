Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a document (like HTML or XML) as a tree of objects.

How do you select an element from the DOM?

You can select an element from the DOM using methods like document.getElementById(), document.querySelector(), or document.getElementsByClassName().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for multiple child elements. It's useful for improving performance and simplifying event management.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using element.setAttribute() and access styles using element.style.propertyName. For example, element.style.backgroundColor = "red";.
